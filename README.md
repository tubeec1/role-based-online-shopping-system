# Role-Based Online Shopping System

## 📌 Description
A full-stack role-based online shopping system developed using PHP (MVC), MySQL, ReactJS, Tailwind CSS, and Redux. It supports Admin, Staff, and Customer roles with permission-based access control.

## 🛠 Tech Stack
- Backend: PHP (MVC)
- Database: MySQL
- Frontend: ReactJS + Tailwind CSS
- State Management: Redux

## 👥 System Roles
- Admin
- Staff
- Customer

## ⚙️ Core Features
- User Authentication (Login / Signup)
- Role & Permission Management (RBAC)
- Category & Product Management
- Product Variants (Images, Colors)
- Order Management System
- Manual Payment Verification
- Dashboard for Admin & Staff

## 🧱 System Architecture
- MVC Backend Structure
- RESTful API (JSON)
- Role-based Middleware
